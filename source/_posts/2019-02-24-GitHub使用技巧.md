# 使用GiThub直接访问html文件
我们先上传文件(这个很简单不再叙述)
然后进入我们需要访问的预览管理页面
![](https://e1sewhere.github.io/images/013.png)

直接在这个页面的url前添加`htmlpreview.github.io/?`访问,就可以访问我们的html了.
![](https://e1sewhere.github.io/images/014.png)
